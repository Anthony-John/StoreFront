Responsive Design

Responsive design is a very simple concept. It simply means that whatever you are designing
will adapt to its environment actively. This concept mainly applies to web design. Today,
there is a vast array of devices and each of them come with there own dimensions, developers
have to be able to make web applications that can adapt their appearance to these devices.
When designing webpages there are many ways to incorporate responsive design into your code.
Some of these options include using certain size units, frameworks, or templates. The two
biggest concepts that govern responsive web design is layout, and size adjustment. Layout,
in responsive design, involves changing the layout of a site’s content according to screen
dimensions. To approach the problem of layout most developers like to apply grid structures
to their web pages that store different objects. To maximize the effectiveness of girds
dynamic sizing units, have to be used. Some units such as em, and percent are good to use when
making responsive web pages because they are scalable and don’t have fixed sizes. In this project
I used the framework Bootstrap to add responsive design to my webpages. In bootstrap there is two
features that I used, cards and grids. Cards allow for a picture to be used with a text box in one
object which is good for keeping things contained. I combined this with bootstrap’s grid feature
so I could control the layout.
